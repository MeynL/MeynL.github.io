组件注册即使用component来创建组件，注册分为局部注册和全局注册两种:

注册组件的注意事项

  • 组件的template(模版)中只能有一个根元素

  • 但用v-if与v-else-if的情况下是允许的

  • data必须是一个函数(防止数据共享)

  • 解析 DOM 模板时的注意事项
    有些 HTML 元素，诸如 <ul>、<ol>、<table> 和 <select>，对于哪些元素可以出现在其内部是有严格限制的。而有些元素，诸如 <li>、<tr> 和 <option>，只能出现在其它某些特定的元素内部。
    因此在渲染时自定义组件在渲染时会被当做无效内容提升到外部，导致渲染错误。但使用：
    1.字符串(例如：template:’…’)
    2.单文件组件(.vue)
    3.<script type="text/template">
    时，将不受此条件约束。
